## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/IamDaleon/onixmedia-internal-app

# Access
$ cd onixmedia-internal-app

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
